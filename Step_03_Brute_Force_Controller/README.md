# Step 03 — Brute-force Controller

Назначение шага: построить эталонный контроллер полного перебора действий.

Brute-force controller перебирает все допустимые действия:

```text
u ∈ [-1.0, 1.0]
num_actions = 61
```

На каждом шаге контроллер:

1. берёт текущее состояние;
2. перебирает 61 возможное действие;
3. для каждого действия делает прогноз на один шаг вперёд;
4. оценивает loss;
5. выбирает действие с минимальным loss;
6. применяет выбранное действие к реальной среде.

Запуск из папки `Step_03_Brute_Force_Controller`:

```powershell
python experiments\run_brute_force_test.py
```

В выводе сравниваются:

```text
open_loop
baseline
brute_force
```
